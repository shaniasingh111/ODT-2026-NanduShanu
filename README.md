18.4 What Changed From the Original Plan 

How did the project change from the initial idea? 

Response: 

[ The project evolved significantly from the initial idea as we began testing and building. The base was originally a simple square, but we changed it to a hexagonal structure to better accommodate the components and improve overall stability and fitting.
